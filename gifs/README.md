# GIFs dos Exercícios

Esta pasta contém os GIFs animados dos exercícios usados no aplicativo IRONFIT.

## Como adicionar GIFs:

1. Baixe GIFs dos exercícios (procure por "exercise GIFs red figure" ou similares)
2. Nomeie os arquivos exatamente como especificado abaixo:
   - supino-reto.gif
   - agachamento.gif
   - levantamento-terra.gif
   - remada-curvada.gif
   - desenvolvimento.gif
   - rosca-direta.gif
   - triceps-pulley.gif
   - puxada-frontal.gif
   - leg-press.gif
   - abdominal-supra.gif
   - elevacao-lateral.gif
   - rosca-martelo.gif

3. Coloque os arquivos nesta pasta (gifs/)

## Fontes recomendadas para GIFs:
- MuscleWiki (musclewiki.com)
- ExRx.net
- Bodybuilding.com
- Procure por "exercise demonstration GIFs"

Os GIFs devem mostrar uma pessoa vermelha/laranja fazendo o exercício corretamente.
